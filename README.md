# PlanFlow - 智能日程管理软件

一款现代化的智能日程管理软件，集任务管理、项目跟踪、日历视图和日记功能于一体。采用玻璃态设计风格，提供优雅的用户体验。

## ✨ 特色功能

- 🎯 **今日概览** - 一目了然的任务统计和日程安排
- 📅 **日历视图** - 可视化月度计划，任务分布清晰
- ✅ **任务管理** - 完整的任务生命周期管理
- 📊 **项目跟踪** - 智能项目进度计算
- 📝 **日记本** - 记录生活点滴，支持心情标记
- 💾 **数据管理** - 本地存储 + 导入导出功能
- 🎨 **现代设计** - 玻璃态UI，流畅动画效果

## 📖 详细使用指南

### 🎯 今日概览
今日概览是软件的主页面，提供当日任务的整体概况。

**主要功能：**
- 📊 **统计卡片**：显示今日任务总数、已完成、进行中、待处理任务数量
- 📝 **今日日程**：列出当天的所有任务安排
- ⚡ **快速操作**：快速添加任务、导出数据

**使用技巧：**
- 统计数据实时更新，完成任务后立即反映
- 点击任务可进行状态切换和删除操作

### 📅 日历视图
月度日历视图，直观显示任务分布。

**主要功能：**
- 🗓️ **月历显示**：清晰的月份视图
- 📋 **任务标记**：有任务的日期会显示任务标签
- ⬅️➡️ **月份切换**：使用左右箭头切换月份
- 📅 **今日高亮**：当前日期特殊标记

**使用方法：**
1. 点击左侧导航"日历视图"
2. 使用 ‹ › 按钮切换月份
3. 鼠标悬停在任务标签上查看详情

### ✅ 任务管理
完整的任务生命周期管理系统。

**创建任务：**
1. 点击"新建任务"按钮
2. 填写任务信息：
   - **任务标题**（必填）
   - **任务描述**（可选）
   - **截止日期**（可选）
   - **优先级**：低/中/高
   - **状态**：待处理/进行中/已完成

**任务操作：**
- ⭕ **切换状态**：待处理 → 进行中 → 已完成
- 🗑️ **删除任务**：点击垃圾桶图标
- 🔍 **筛选任务**：使用筛选按钮过滤显示

**任务筛选：**
- **全部**：显示所有任务
- **待处理**：仅显示未开始的任务
- **进行中**：仅显示正在进行的任务
- **已完成**：仅显示已完成的任务
- **高优先级**：仅显示高优先级任务

### 📊 项目管理
跟踪长期项目的进度和状态。

**创建项目：**
1. 点击"新建项目"按钮
2. 填写项目信息：
   - **项目名称**（必填）
   - **项目描述**（可选）
   - **预计完成日期**（可选）

**项目特性：**
- 📈 **自动进度计算**：根据相关任务自动计算完成进度
- 🎯 **进度可视化**：进度条直观显示项目状态
- 📅 **截止日期提醒**：显示项目截止时间

**进度计算规则：**
- 系统会自动查找描述中包含项目名称的任务
- 进度 = 已完成任务数 ÷ 总相关任务数 × 100%

### 📝 日记本
记录生活点滴，支持心情标记。

**写日记：**
1. 点击"写日记"按钮
2. 填写日记内容：
   - **日记标题**（必填）
   - **日记内容**（详细记录）
   - **当日心情**：😊开心/😐一般/😢伤心/🤩兴奋/😴疲惫

**日记特性：**
- 📅 **自动时间戳**：记录创建时间
- 🔄 **时间排序**：按创建时间倒序显示
- 😊 **心情统计**：通过表情符号记录心情
- 📖 **内容预览**：长内容自动截断显示

**使用建议：**
- 不需要每天都写，想记录时随时添加
- 可以记录重要事件、感悟、计划等
- 利用心情标记追踪情绪变化

### 💾 数据管理

**自动保存：**
- 所有数据自动保存到浏览器本地存储
- 无需手动保存操作
- 数据在浏览器中持久保存

**数据导出：**
1. 点击任意页面的"导出数据"按钮
2. 系统自动生成JSON格式文件
3. 文件命名格式：`planflow-data-YYYY-MM-DD.json`
4. 文件包含所有任务、项目、日记数据

**数据导入：**
1. 点击侧边栏的"📥 导入数据"按钮
2. 选择之前导出的JSON文件
3. 点击"导入"按钮
4. 系统验证数据格式并导入

**数据备份建议：**
- 定期导出数据作为备份
- 更换设备时使用导入功能迁移数据
- 重要数据可保存多个备份副本

## 🛠️ 故障排除

### 常见问题

**Q: 数据丢失了怎么办？**
A: 
1. 检查是否在同一浏览器中打开
2. 清除浏览器缓存可能导致数据丢失
3. 使用之前导出的备份文件导入数据

**Q: 导入数据失败？**
A:
1. 确保文件格式为JSON
2. 确保文件是从本软件导出的
3. 检查文件是否损坏

**Q: 界面显示异常？**
A:
1. 刷新页面（Ctrl+F5 或 Cmd+Shift+R）
2. 检查浏览器版本是否支持
3. 尝试其他浏览器

**Q: 任务在日历中不显示？**
A:
1. 确保任务设置了截止日期
2. 检查日历是否在正确的月份
3. 刷新页面重新加载数据

### 数据恢复
如果数据意外丢失：
1. 检查浏览器的本地存储是否被清理
2. 使用最近的备份文件恢复数据
3. 重新创建重要的任务和项目
